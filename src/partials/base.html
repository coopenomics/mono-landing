Базовая поставка

Включено в каждый
кооператив

Четыре приложения в комплекте — достаточно для ведения полноценной кооперативной деятельности. Дополнительные подключаются из каталога.

БАЗОВОЕ / 01

Регистрация пайщиков

Удалённое вступление онлайн, выдача электронной подписи, ведение электронного реестра пайщиков. Физлица, ИП, организации. Импорт офлайн-пайщиков в систему.

БАЗОВОЕ / 02

Решения совета

Автоматическое объявление собрания совета по стандартным вопросам. Ручное объявление собрания по произвольным вопросам. Голосование онлайн. Протокол формируется автоматически — без бумаги.

БАЗОВОЕ / 03

Общие собрания

Очередные и внеочередные общие собрания пайщиков. Рассылка уведомлений, сбор подписей и голосов, формирование протокола за подписью председателя и секретаря собрания. Собрания переобъявляются до принятия решения по кворуму.

БАЗОВОЕ / 04

Цифровой кошелёк

Лицевой счёт пайщика и кооператива. Учёт взносов, возвратов и балансов по всем программам. Приём платежей через ЮКасса или банковским переводом по QR-коду.
