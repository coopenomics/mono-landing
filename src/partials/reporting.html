Учёт и отчётность

Первичные документы —
основа для отчётности

Все документы, формируемые системой, являются первичными учётными документами. На их основе строится отчётность для ФНС и ФСС.

Система автоматически формирует отчетные документы в форматах контролирующих органов. Кооперативы получают готовые отчеты для импорта в 1С и/или отправки их в электронном виде через СКБ-Контур.

Отчёты для ФНС и ФСС

Регуляторная отчётность формируется автоматически на основе реестра операций. Готова к сдаче без ручной сборки.

  • Автоматическая сборка из реестра операций
  • Выгрузка в форматах контролирующих органов
  • История операций по всем счетам
